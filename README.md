# LAMPTutorial
A basic tutorial on how to install LAMP on Ubuntu and Ubuntu-based Linux distros

## What is LAMP?
The LAMP stack is an acronym for a set of open-source software that can be used to create websites and web applications.

It stands for:

..* *L* inux 
..* *A* pache
..* *M* MySql
..* *P* HP
